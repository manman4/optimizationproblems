# Optimization Problems in Mathematics

A curated collection of optimization problems in mathematics, focusing on problems where there is some constant C arising from an optimization problem - for instance, the best constant for which a certain inequality is true.

## Purpose

This repository aims to:
- Record known upper and lower bounds for mathematical constants arising from optimization problems
- Track progress on open problems in optimization and inequalities
- Encourage crowdsourced contributions and updates via pull requests
- Provide a centralized, accessible reference for researchers

## Structure

Each optimization problem is documented in its own markdown file in the `problems/` directory. The README provides an index of all problems.

## Contributing

We welcome contributions! To add a new problem or update bounds:

1. Fork this repository
2. Create or update a problem file in the `problems/` directory (see [template.md](template.md))
3. Update this README to include a link to your problem
4. Submit a pull request with:
   - A clear description of the changes
   - Citations or references for any new bounds
   - Any relevant context or discussion

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed guidelines.

## Problem Format

Each problem file should include:
- **Problem Statement**: Clear mathematical description
- **Current Best Bounds**: Known upper and lower bounds with references
- **History**: Timeline of improvements
- **References**: Academic papers and resources
- **Notes**: Additional context, conjectures, or open questions

See [template.md](template.md) for the standard format.
